# Estudos Java & Programação Orientada a Objetos

Repositório dedicado à consolidação de conceitos de Programação Orientada a Objetos (POO), boas práticas de mercado e arquitetura no ecossistema Java.

---

## 📂 Organização do Portfólio

O repositório é segmentado em módulos estruturados para refletir a evolução cronológica dos estudos:

*   **`01-curso-guanabara/`**: Exercícios de base em POO e projeto final de streaming de vídeo refatorado com técnicas de *Clean Code*.
*   **`02-curso-nelio-alves/`**: Projetos avançados e conceitos corporativos desenvolvidos durante o curso da Udemy (Tratamento de exceções, Enumerações, Composição e Padrões de Projeto).
